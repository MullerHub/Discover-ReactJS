Tipos de tipagem:

10, 15, 44.... 90: number

"10", "15", "44".... "90": string

Tipagem em Generics:

useState<T>   "foca no <T>"

Uma boa pratica é usar letras padronizadas, mas não é obrigatorio como

S => STATE / estado
T => TYPE / TIPO
K => KEY / CHAVE
V => VALUE / VALOR
E => ELEMENT / ELEMENTO
